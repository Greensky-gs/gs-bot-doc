## Commande pin
Épingle un message dans le salon de la commande.

## Catégorie
[Modération](../categories/moderation.md)

## Permissions
`Gérer les messages`

## Cooldown
**5** secondes

## Alias
`epingle`, `épingle`

## Arguments
1. Identifiant du message à épingler ( [identifiant](../others/id.md) )
