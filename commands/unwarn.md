## Commande unwarn
Enlève un avertissement à un utilsateur

## Catégorie
[REMPLIR](../categories/REMPLIR.md)

## Permissions
`Gérer le serveur`

## Cooldown
**5** secondes

## Alias
`dewarn`

## Arguments
1. Utilisateur ( [utilisateur](../others/user.md) )
2. Identifiant du warn ( visible via la commande [`modlogs`](./modlgos.md) )
