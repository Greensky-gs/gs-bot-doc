## Commande censor
Censure le pseudo d'un membre

## Catégorie
[Modération](../categories/moderation.md)

## Permissions
`Gérer le serveur`

## Cooldown
**5** secondes

## Alias
*Pas d'alias*

## Arguments
1. Un membre ( [utilisateur](../others/user.md) )
2. Une raison ( [texte](../others/texte.md) )
